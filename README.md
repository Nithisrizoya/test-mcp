# Even or Odd Predictor

A simple Python program to determine whether a given number is even or odd.

## Requirements

### System Requirements
- **Python 3.6+** (Any modern Python version)
- **Operating System**: Windows, macOS, or Linux
- **Terminal/Command Prompt**: To run the script

### Python Dependencies
- **None** - This project uses only Python built-in modules
  - No external packages required
  - No pip installations needed

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Nithisrizoya/test-mcp.git
cd test-mcp
```

2. No additional setup needed - all dependencies are built-in!

## Usage

Run the script:
```bash
python main.py
```

Or use Python 3 explicitly:
```bash
python3 main.py
```

Example:
```
Even or Odd Predictor
----------------------------------------
Enter a number: 5
5 is Odd
```

## How it Works

The program:
1. Takes user input for a number
2. Uses the modulo operator (%) to check divisibility by 2
3. Returns whether the number is Even or Odd
4. Includes error handling for invalid inputs

## Features

- ✅ Simple and lightweight
- ✅ No external dependencies
- ✅ Error handling for non-integer inputs
- ✅ Clear user interface
